# Zewadi - Premium Digital Experience

This is a [Next.js](https://nextjs.org) project built for the Zewadi brand, featuring modern animations with GSAP and a high-fidelity design system.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: GSAP (GreenSock Animation Platform)
- **Icons**: Lucide React
